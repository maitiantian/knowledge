# PEP8: Style Guide for Python Code

[PEP8](https://www.python.org/dev/peps/pep-0008/)


## Code Lay-out

* #### Indentation
    Use 4 spaces per indentation level.
* #### Tabs or Spaces?
    Spaces are the preferred indentation method.
* #### Maximum Line Length
    Limit all lines to a maximum of 79 characters.
